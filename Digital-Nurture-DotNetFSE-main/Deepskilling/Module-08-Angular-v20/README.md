# Module 08 – Angular (v20.0)

## Overview

Angular is a TypeScript-based front-end framework developed by Google for building scalable, single-page web applications (SPAs). This module introduces Angular architecture, components, directives, forms, dependency injection, routing, HTTP communication, state management, RxJS, and testing.

## Learning Objectives

After completing this module, you will be able to:

- Explain Angular architecture and project structure.
- Set up Angular using Angular CLI.
- Create reusable Angular components.
- Implement data binding and lifecycle hooks.
- Use directives and pipes.
- Build template-driven and reactive forms.
- Implement dependency injection and services.
- Configure routing and navigation.
- Consume REST APIs using HttpClient.
- Handle asynchronous programming with RxJS.
- Manage application state using Services and NgRx.
- Test Angular applications using Jasmine and Karma.
