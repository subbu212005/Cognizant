# Exercise 4 - PIVOT and UNPIVOT

## Objective

Transform data for reporting purposes using pivot and unpivot techniques.

## Goal

Display monthly sales quantity per product in a pivoted format and convert it back into row format.

## Scenario

The Online Retail Store wants monthly sales reports showing products as rows and months as columns.

## Steps

1. Aggregate sales quantity by Product and Month.
2. Transform rows into columns (Pivot).
3. Convert columns back into rows (Unpivot).

## Concepts Used

* Data Transformation
* Pivoting Data
* Unpivoting Data
* Aggregate Functions
* CASE Statements

## Observation

The pivot operation transformed row-based sales data into a report-friendly format, while the unpivot operation restored the original structure.

## Learning Outcome

Learned how data transformation techniques improve reporting and analysis by converting rows into columns and vice versa.

## Conclusion

Pivoting and unpivoting are useful reporting techniques that help present data in different analytical formats.
