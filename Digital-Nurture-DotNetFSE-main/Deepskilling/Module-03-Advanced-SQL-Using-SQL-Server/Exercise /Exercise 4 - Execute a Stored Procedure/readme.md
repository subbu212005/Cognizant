# Exercise 4 - Execute a Stored Procedure

## Objective

Execute a stored procedure and retrieve data from the database.

## Goal

Understand how to execute stored procedures and reuse database logic.

## Scenario

The Online Retail Store has a stored procedure that retrieves all product details. Execute the procedure and view the results.

## Concepts Used

* Stored Procedures
* EXEC Statement
* Query Reusability

## Observation

The stored procedure executed successfully and returned product information from the database.

## Learning Outcome

Learned how to execute stored procedures and reuse predefined SQL logic for efficient database operations.

## Conclusion

Stored procedures simplify database management by centralizing reusable business logic.
