# Exercise 5 - Return Data From Stored Procedure

## Objective

Create and execute a Stored Procedure that returns data from a database table.

## Goal

Retrieve product information using a Stored Procedure.

## Scenario

The Online Retail Store requires a reusable Stored Procedure to fetch all product details whenever requested.

## Steps

1. Create the Products table.
2. Insert sample product records.
3. Create a Stored Procedure to retrieve all products.
4. Execute the Stored Procedure.
5. Display the returned data.

## Concepts Used

* Stored Procedure
* CREATE PROCEDURE
* SELECT Statement
* EXEC Statement

## Observation

The Stored Procedure successfully returned all records from the Products table without rewriting the SQL query.

## Learning Outcome

Learned how Stored Procedures return data, improve code reusability, simplify maintenance, and centralize business logic.

## Conclusion

Returning data through Stored Procedures makes SQL code reusable, secure, and easier to maintain in enterprise database applications.
