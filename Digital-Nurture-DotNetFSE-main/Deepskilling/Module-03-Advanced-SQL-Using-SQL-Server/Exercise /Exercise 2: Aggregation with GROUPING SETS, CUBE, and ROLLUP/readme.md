# Exercise 2 - Aggregation with GROUPING SETS, CUBE, and ROLLUP

## Objective

Analyze sales data across multiple dimensions using advanced SQL aggregation techniques.

## Goal

Generate reports showing total quantity sold by Region and Category using GROUPING SETS, ROLLUP, and CUBE.

## Scenario

The online retail store wants to analyze sales performance across different regions and product categories.

## Steps

1. Join Orders, OrderDetails, Customers, and Products tables.
2. Use GROUPING SETS to get totals by Region, Category, and both.
3. Use ROLLUP to generate subtotals and grand totals.
4. Use CUBE to generate all possible combinations of Region and Category.

## Concepts Used

* GROUPING SETS
* ROLLUP
* CUBE
* Aggregate Functions
* JOIN Operations

## Observation

GROUPING SETS generated customized summary reports, ROLLUP produced hierarchical subtotals and grand totals, and CUBE generated all possible aggregations across dimensions.

## Learning Outcome

Learned how to perform multidimensional data analysis and generate summary reports using GROUPING SETS, ROLLUP, and CUBE.

## Conclusion

This exercise demonstrated advanced aggregation techniques that help businesses analyze sales performance efficiently across multiple dimensions.
