# Exercise 6 - Return Data From Scalar Function

## Objective

Create and execute a Scalar User-Defined Function (UDF) to return a calculated value.

## Goal

Understand how Scalar Functions return a single value based on input parameters.

## Scenario

The Online Retail Store wants to calculate the discounted price of a product using a reusable Scalar Function.

## Steps

1. Create a Products table.
2. Insert sample product records.
3. Create a Scalar Function to calculate the discounted price.
4. Execute the function using a SELECT statement.
5. Display the calculated result.

## Concepts Used

* Scalar Function
* CREATE FUNCTION
* RETURN Statement
* Parameters
* SELECT Statement

## Observation

The Scalar Function returned a single calculated value based on the input parameter and simplified reusable business logic.

## Learning Outcome

Learned how Scalar Functions encapsulate reusable calculations and return a single value that can be used in SQL queries.

## Conclusion

Scalar Functions improve code reusability, readability, and maintainability by centralizing frequently used calculations.
