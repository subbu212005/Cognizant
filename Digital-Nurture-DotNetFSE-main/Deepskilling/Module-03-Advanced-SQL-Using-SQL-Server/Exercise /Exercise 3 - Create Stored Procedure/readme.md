# Exercise 3 - Create Stored Procedure

## Objective

Create a Stored Procedure to retrieve product information from the database.

## Goal

Understand how to create and use Stored Procedures in SQL Server.

## Scenario

The Online Retail Store frequently needs to retrieve product details. To avoid writing the same query repeatedly, a Stored Procedure is created.

## Concepts Used

* Stored Procedure
* CREATE PROCEDURE
* SELECT Statement

## Steps

1. Create a Products table.
2. Insert sample records.
3. Create a Stored Procedure.
4. Execute the Stored Procedure.

## Observation

The Stored Procedure successfully stored reusable SQL logic and returned product information when executed.

## Learning Outcome

Learned how to create Stored Procedures to improve code reusability, maintainability, and database performance.

## Conclusion

Stored Procedures help centralize business logic and reduce repetitive SQL code.
