## Result

The query successfully displayed the top 3 most expensive products in each category using ROW_NUMBER(), RANK(), and DENSE_RANK().

## Observation

* ROW_NUMBER() assigned a unique rank to each product within its category.
* RANK() assigned ranks based on product prices and would skip rank values if duplicate prices existed.
* DENSE_RANK() assigned consecutive ranks without gaps.
* PARTITION BY Category ensured ranking restarted for each category.
