# SQL Server Fundamentals

## Introduction

Microsoft SQL Server is a Relational Database Management System (RDBMS) developed by Microsoft. It is used to store, manage, retrieve, and analyze data efficiently.

SQL Server supports enterprise-level applications and provides high performance, security, and scalability.

---

## SQL Server Editions

### Express Edition

* Free version
* Suitable for learning and small applications

### Developer Edition

* Free for development and testing
* Includes all Enterprise features

### Standard Edition

* Suitable for medium-sized organizations
* Provides core database features

### Enterprise Edition

* Designed for large-scale applications
* Advanced security and high availability features

---

## Key Components of SQL Server

### Database Engine

Stores, processes, and secures data.

### SQL Server Agent

Used for scheduling jobs and automation tasks.

### Integration Services (SSIS)

Used for data migration and ETL operations.

### Analysis Services (SSAS)

Used for data analysis and business intelligence.

### Reporting Services (SSRS)

Used for generating reports and dashboards.

---

## SQL Server Instances

A SQL Server Instance is an installation of SQL Server running on a server.

### Default Instance

* Identified only by server name.

Example:

```text
localhost
```

### Named Instance

* Identified by server name and instance name.

Example:

```text
localhost\SQLEXPRESS
```

---

## SQL Server Management Studio (SSMS)

SQL Server Management Studio (SSMS) is the graphical tool used to manage SQL Server databases.

### Features of SSMS

* Create databases
* Execute SQL queries
* Manage tables
* Create stored procedures
* Monitor server performance
* Manage security and permissions

---

## Advantages of SQL Server

* High performance
* Strong security
* Backup and recovery support
* Scalability
* Business intelligence support
* Cloud integration
