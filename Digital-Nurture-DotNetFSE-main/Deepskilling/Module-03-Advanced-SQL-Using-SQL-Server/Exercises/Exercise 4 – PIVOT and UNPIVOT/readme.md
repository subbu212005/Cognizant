# Exercise 4 – PIVOT and UNPIVOT

## Objective

The objective of this exercise is to transform relational data into a report-friendly format using PIVOT and convert it back into its original structure using UNPIVOT.

## SQL Concepts Used

- PIVOT
- UNPIVOT
- SUM()
- MONTH()
- GROUP BY

## Tasks Performed

- Calculated monthly sales quantity for each product.
- Converted row-based data into column-based format using PIVOT.
- Converted the pivoted table back into rows using UNPIVOT.
- Compared both data representations for reporting purposes.

## Learning Outcomes

- Learned how to reshape SQL query results.
- Understood the practical use of PIVOT for business reports.
- Learned how UNPIVOT restores normalized data.
- Improved SQL reporting and visualization skills.

## Conclusion

PIVOT and UNPIVOT provide flexible ways to transform data for dashboards, reports, and business intelligence applications.
