# Exercise 5 – Using Common Table Expressions (CTEs)

## Objective

The objective of this exercise is to simplify complex SQL queries using Common Table Expressions (CTEs) by identifying customers who have placed more than three orders.

## SQL Concepts Used

- WITH
- Common Table Expression (CTE)
- COUNT()
- GROUP BY
- JOIN
- WHERE

## Tasks Performed

- Created a CTE to count the total number of orders placed by each customer.
- Joined the CTE with the Customers table.
- Filtered customers who placed more than three orders.
- Displayed customer information along with total order count.

## Learning Outcomes

- Learned how CTEs improve SQL query readability.
- Simplified complex aggregation queries.
- Improved understanding of query organization and maintainability.
- Gained experience using CTEs for analytical queries.

## Conclusion

Common Table Expressions make SQL queries easier to read, maintain, and reuse. They are especially useful when working with complex analytical and reporting queries.
