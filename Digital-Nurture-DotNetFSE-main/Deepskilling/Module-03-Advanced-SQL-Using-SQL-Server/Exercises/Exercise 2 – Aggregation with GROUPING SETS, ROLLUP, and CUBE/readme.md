# Exercise 2 – Aggregation with GROUPING SETS, ROLLUP, and CUBE

## Objective

The objective of this exercise is to analyze sales data using advanced SQL aggregation techniques across multiple dimensions such as Region and Product Category.

## SQL Concepts Used

- GROUP BY
- SUM()
- GROUPING SETS
- ROLLUP
- CUBE
- JOIN

## Tasks Performed

- Joined Customers, Orders, OrderDetails, and Products tables.
- Calculated total quantity sold by Region and Category.
- Generated custom summaries using `GROUPING SETS`.
- Produced hierarchical subtotals and grand totals using `ROLLUP`.
- Generated all possible aggregation combinations using `CUBE`.

## Learning Outcomes

- Learned advanced aggregation techniques.
- Understood the differences between `GROUPING SETS`, `ROLLUP`, and `CUBE`.
- Generated multidimensional reports efficiently.
- Improved SQL reporting and data summarization skills.

## Conclusion

Advanced aggregation functions simplify complex reporting requirements by generating summaries, subtotals, and grand totals with minimal SQL code.
