# Lab 6 - Updating and Deleting Records

## Objective

Learn how to update and delete records using Entity Framework Core.

---

## Methods Used

- FirstOrDefaultAsync()
- SaveChangesAsync()
- Remove()

---

## Run

```bash
dotnet run
```

---

## Expected Output

```
===== UPDATE PRODUCT =====

Laptop price updated successfully.

===== DELETE PRODUCT =====

Rice Bag deleted successfully.

===== REMAINING PRODUCTS =====

1 Laptop ₹70000
```

---

## Conclusion

This lab demonstrates updating existing records and deleting records from the SQL Server database using Entity Framework Core.
