# Module 4 - NUnit and Moq

## Overview

This module introduces automated testing concepts using NUnit and Moq frameworks in .NET. Automated testing helps developers verify software functionality quickly and reliably.

NUnit is a popular unit testing framework used to write and execute test cases, while Moq is used to create mock objects for testing components that depend on external services.

---

## Learning Objectives

After completing this module, learners will be able to:

* Understand automated testing and its benefits.
* Learn Test Driven Development (TDD).
* Write unit tests using NUnit.
* Test methods, exceptions, arrays, and collections.
* Understand dependency injection.
* Create mock objects using Moq.
* Test interactions between software components.
