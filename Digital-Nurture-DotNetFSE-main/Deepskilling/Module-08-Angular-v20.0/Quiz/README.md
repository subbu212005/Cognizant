# Angular Quiz Collection

This folder contains Angular quiz questions and reference materials collected from multiple learning platforms. These quizzes help reinforce Angular fundamentals, including components, data binding, directives, services, routing, lifecycle hooks, forms, HTTP communication, and standalone components.

## Files Included

### 📄 Angular-Quiz.md
- Comprehensive Angular quiz with multiple-choice questions.
- Includes correct answers and explanations.
- Covers Angular basics, CLI, components, directives, routing, services, lifecycle hooks, forms, and HTTP.

### 📄 Java Guides angular.md
- Angular quiz based on Java Guides.
- Contains 30 multiple-choice questions with answers and explanations.
- Useful for interview preparation and concept revision.

### 🖼️ angular w3.png
- Screenshot of Angular quiz results from W3Schools.
- Serves as proof of quiz completion.

## Learning Outcomes

After completing these quizzes, learners will be able to:

- Understand Angular architecture and project structure.
- Create Angular applications using Angular CLI.
- Build reusable components and directives.
- Implement data binding techniques.
- Use structural and attribute directives effectively.
- Create and consume Angular services.
- Perform HTTP requests using HttpClient.
- Implement routing and navigation.
- Apply lifecycle hooks appropriately.
- Work with forms and two-way data binding.
- Understand standalone components and modern Angular features.
- Prepare for Angular assessments and technical interviews.
