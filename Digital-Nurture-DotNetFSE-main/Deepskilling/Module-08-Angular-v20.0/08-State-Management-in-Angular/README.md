# Module 08 – State Management in Angular

## Overview

State Management is a technique used to manage, store, and share data across different components in an Angular application. As applications grow larger, maintaining consistent data becomes increasingly important. Angular offers multiple approaches for managing application state, from simple service-based solutions to advanced libraries like NgRx.

This module introduces the fundamentals of State Management, demonstrates how Angular Services can be used to share data between components, and explores NgRx for building scalable, predictable, and maintainable enterprise applications.

---

## Learning Objectives

After completing this module, you will be able to:

- Understand State Management in Angular.
- Differentiate between local and shared state.
- Share data using Angular Services.
- Understand NgRx architecture.
- Work with Store, Actions, Reducers, Effects, and Selectors.
- Build scalable Angular applications.
