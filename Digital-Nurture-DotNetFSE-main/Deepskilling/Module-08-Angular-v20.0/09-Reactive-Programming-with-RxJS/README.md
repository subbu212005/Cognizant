# Module 09 – Reactive Programming with RxJS

## Overview

Reactive Programming is a programming paradigm that focuses on handling asynchronous data streams and events. Angular uses RxJS (Reactive Extensions for JavaScript) extensively to manage asynchronous operations such as HTTP requests, user interactions, routing, and real-time data updates.

RxJS provides powerful features like Observables, Operators, and Subjects, enabling developers to write clean, maintainable, and scalable applications.

This module introduces the fundamentals of RxJS, explores commonly used operators, and demonstrates reactive programming patterns in Angular.
