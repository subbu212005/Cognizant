# Event Binding in Angular

## Overview

Event Binding is a data binding technique in Angular that allows communication from the **View (HTML)** to the **Component (TypeScript)**. It is used to execute component methods when a user performs an action such as clicking a button, typing in a textbox, or moving the mouse.

Event Binding is represented using **parentheses `( )`**.

---

# What is Event Binding?

Event Binding listens for events generated by HTML elements and calls a method in the component class.

### Syntax

```html
(event)="method()"
```

Example:

```html
<button (click)="showMessage()">
    Click Me
</button>
```

When the button is clicked, Angular executes the `showMessage()` method.

---

# Common Events

| Event         | Description          |
| ------------- | -------------------- |
| `(click)`     | Button click         |
| `(keyup)`     | Key released         |
| `(keydown)`   | Key pressed          |
| `(change)`    | Value changed        |
| `(mouseover)` | Mouse enters element |
| `(mouseout)`  | Mouse leaves element |
| `(submit)`    | Form submission      |

# Example

## student.ts

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-student',
  standalone: true,
  imports: [],
  templateUrl: './student.html',
  styleUrl: './student.css'
})
export class StudentComponent {

  message = "";

  showMessage() {
    this.message = "Welcome to Angular Event Binding!";
  }

}
```

---

## student.html

```html
<h2>Event Binding Example</h2>

<button (click)="showMessage()">
    Click Me
</button>

<p>{{ message }}</p>
```

---

## student.css

```css
h2{
    color: green;
}

button{
    padding:10px 20px;
    margin-top:15px;
    cursor:pointer;
}

p{
    font-weight:bold;
    color:blue;
}
```

# Difference Between Property Binding and Event Binding

| Property Binding        | Event Binding           |
| ----------------------- | ----------------------- |
| Component → View        | View → Component        |
| Uses `[ ]`              | Uses `( )`              |
| Updates HTML properties | Responds to user events |
