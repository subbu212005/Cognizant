# Module 10 – Testing Angular Applications

## Overview

Testing is an essential part of Angular development that helps ensure applications are reliable, maintainable, and free from defects. Angular provides a comprehensive testing framework using Jasmine and Karma for unit testing, while component and service testing verify the functionality of individual parts of an application. End-to-End (E2E) testing validates complete user workflows.

This module introduces Angular's testing tools and demonstrates how to write and execute tests for components and services.

---

- Write Unit Tests using Jasmine.
- Execute tests using Karma.
- Test Angular Components.
