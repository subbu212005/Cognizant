# Module 06 – Routing and Navigation

## Overview

Routing is one of the most important features of Angular. It enables navigation between different views or components without reloading the entire web page. Angular applications are Single Page Applications (SPAs), where routing provides a seamless user experience by dynamically loading components based on the URL.

This module introduces Angular Routing, route configuration, navigation using RouterLink and RouterOutlet, route parameters, and lazy loading for improving application performance.

---

After completing this module, you will be able to:

- Understand Angular Routing.
- Configure application routes.
- Navigate between components.

# Advantages of Angular Routing

- Enables seamless navigation.
- Eliminates full page reloads.
- Improves application performance.
- Supports browser history.
- Provides clean URL management.
- Enhances user experience.

---

# Advantages of Lazy Loading

- Faster application startup.
- Smaller initial bundle size.
- Better memory utilization.
- Improved scalability.
- Loads modules only when required.
