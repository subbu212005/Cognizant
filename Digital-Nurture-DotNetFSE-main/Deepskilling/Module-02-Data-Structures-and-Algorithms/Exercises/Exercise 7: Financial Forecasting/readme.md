# Exercise 7 - Financial Forecasting

## Objective

Implement a simple financial forecasting model using recursion.

## Concepts Used

* Recursion
* Time Complexity Analysis
* Mathematical Computation

## Description

Financial forecasting helps estimate future values based on current data and growth rates.

This exercise calculates future investment value using recursive growth calculation.

## Program Output

```text

```

## Observation

The recursive function calculates future value by repeatedly applying the annual growth rate.

## Learning Outcome

Financial forecasting techniques help organizations make informed decisions by estimating future outcomes based on current trends.
