# Exercise 2 - E-commerce Platform Search Function

## Objective

Implement a search function for an e-commerce platform using searching algorithms.

## Concepts Used

* Arrays
* Linear Search
* Binary Search
* Time Complexity Analysis

## Description

E-commerce platforms contain thousands of products. Searching algorithms help users quickly locate products based on product IDs or names.

This exercise demonstrates searching for a product using Linear Search.

## Program Output

```text
Product Found: Laptop
Position: 2
```

## Observation

The search algorithm successfully located the required product. Linear Search checks elements sequentially until the target product is found.

## Learning Outcome

After completing this exercise, I understood how searching algorithms are applied in real-world e-commerce applications for product retrieval.

## Conclusion

Searching algorithms improve the efficiency of product discovery and enhance user experience on e-commerce platforms.
