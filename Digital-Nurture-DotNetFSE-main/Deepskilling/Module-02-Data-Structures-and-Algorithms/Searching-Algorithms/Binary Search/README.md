# Binary Search

## Introduction

Binary Search is an efficient searching algorithm used on sorted arrays.

It repeatedly divides the search space into two halves until the target element is found.

## Working

1. Find the middle element.
2. Compare it with the target.
3. If equal, return the position.
4. If target is smaller, search left half.
5. If target is larger, search right half.

## Time Complexity

| Case         | Complexity |
| ------------ | ---------- |
| Best Case    | O(1)       |
| Average Case | O(log n)   |
| Worst Case   | O(log n)   |

## Advantages

* Faster than Linear Search
* Efficient for large sorted datasets

## Learning Outcome

Learned how Binary Search reduces search time using divide-and-conquer strategy.
