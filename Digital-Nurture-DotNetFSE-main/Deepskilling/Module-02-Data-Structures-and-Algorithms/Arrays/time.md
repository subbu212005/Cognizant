## Time Complexity

| Operation | Time Complexity |
|-----------|----------------|
| Access | O(1) |
| Traversal | O(n) |
| Search | O(n) |
| Insertion | O(n) |
| Deletion | O(n) |

### Explanation

- **Access : O(1)**  
  Array elements can be accessed directly using their index.

- **Traversal : O(n)**  
  Every element is visited once.

- **Search : O(n)**  
  Linear search may need to check all elements.

- **Insertion : O(n)**  
  Elements may need to be shifted to create space.

- **Deletion : O(n)**  
  Elements may need to be shifted after deletion.

## Observation

Array traversal visits each element exactly once, resulting in O(n) time complexity. Searching, insertion, and deletion operations may require processing multiple elements, making them O(n) operations.
