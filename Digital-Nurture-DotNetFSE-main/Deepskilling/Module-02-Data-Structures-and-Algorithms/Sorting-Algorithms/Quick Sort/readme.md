# Quick Sort

## Introduction

Quick Sort is a divide-and-conquer sorting algorithm that selects a pivot element and partitions the array around it.

## Working

1. Select a pivot element.
2. Partition the array into smaller and larger elements.
3. Recursively sort both partitions.
4. Combine the results.

## Time Complexity

| Case         | Complexity |
| ------------ | ---------- |
| Best Case    | O(n log n) |
| Average Case | O(n log n) |
| Worst Case   | O(n²)      |

## Advantages

* Very fast in practice
* Efficient for large datasets
* Uses divide-and-conquer approach

## Learning Outcome

Learned how partitioning and recursion are used to sort data efficiently.
