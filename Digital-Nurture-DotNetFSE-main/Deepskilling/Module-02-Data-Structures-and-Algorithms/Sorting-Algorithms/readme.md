# Sorting Algorithms

## Introduction

Sorting is the process of arranging data in a specific order, typically ascending or descending. Sorting algorithms improve the efficiency of searching, data processing, and analysis.

## Why Sorting Algorithms?

Sorting algorithms help to:

* Organize data efficiently.
* Improve searching performance.
* Simplify data analysis.
* Enhance application performance.
* Support efficient data management.

## Types of Sorting Algorithms Covered

### Bubble Sort

Repeatedly compares adjacent elements and swaps them if they are in the wrong order.

### Insertion Sort

Builds the sorted array one element at a time by inserting elements into their correct positions.

### Heap Sort

Uses a binary heap data structure to sort elements efficiently.

### Quick Sort

Uses the divide-and-conquer approach by selecting a pivot element and partitioning the array.

### Merge Sort

Divides the array into smaller subarrays, sorts them, and merges them back together.

## Time Complexity Comparison

| Algorithm      | Best Case  | Average Case | Worst Case |
| -------------- | ---------- | ------------ | ---------- |
| Bubble Sort    | O(n)       | O(n²)        | O(n²)      |
| Insertion Sort | O(n)       | O(n²)        | O(n²)      |
| Heap Sort      | O(n log n) | O(n log n)   | O(n log n) |
| Quick Sort     | O(n log n) | O(n log n)   | O(n²)      |
| Merge Sort     | O(n log n) | O(n log n)   | O(n log n) |
