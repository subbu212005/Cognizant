# Merge Sort

## Introduction

Merge Sort is a divide-and-conquer sorting algorithm that divides the array into smaller subarrays, sorts them, and merges them back together.

## Working

1. Divide the array into two halves.
2. Recursively sort both halves.
3. Merge the sorted halves.
4. Repeat until the entire array is sorted.

## Time Complexity

| Case         | Complexity |
| ------------ | ---------- |
| Best Case    | O(n log n) |
| Average Case | O(n log n) |
| Worst Case   | O(n log n) |

## Advantages

* Stable sorting algorithm
* Consistent performance
* Efficient for large datasets

## Learning Outcome

Learned how recursive division and merging help achieve efficient sorting performance.
