# Bubble Sort

## Introduction

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order.

## Time Complexity

| Case         | Complexity |
| ------------ | ---------- |
| Best Case    | O(n)       |
| Average Case | O(n²)      |
| Worst Case   | O(n²)      |

## Learning Outcome

Learned the basic concept of sorting using adjacent element comparisons.
