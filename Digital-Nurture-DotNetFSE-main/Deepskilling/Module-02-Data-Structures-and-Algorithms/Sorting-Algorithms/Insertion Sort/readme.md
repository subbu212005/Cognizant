# Insertion Sort

## Introduction

Insertion Sort builds a sorted portion of the array by inserting elements into their correct positions.

## Time Complexity

| Case         | Complexity |
| ------------ | ---------- |
| Best Case    | O(n)       |
| Average Case | O(n²)      |
| Worst Case   | O(n²)      |

## Learning Outcome

Learned how elements are inserted into their correct position within a sorted sequence.
