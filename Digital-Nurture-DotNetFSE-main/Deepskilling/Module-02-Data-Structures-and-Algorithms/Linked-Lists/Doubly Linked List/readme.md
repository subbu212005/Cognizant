# Doubly Linked List

## Introduction

A Doubly Linked List contains data and two references: one to the previous node and one to the next node.

Example:

```text
NULL ← 10 ⇄ 20 ⇄ 30 → NULL
```

## Operations

* Traversal
* Search
* Insert
* Delete

## Time Complexity

| Operation | Time Complexity |
| --------- | --------------- |
| Traversal | O(n)            |
| Search    | O(n)            |
| Insert    | O(1) or O(n)    |
| Delete    | O(1) or O(n)    |

## Advantages

* Bidirectional traversal
* Easier deletion operations

## Learning Outcome

Learned how previous and next pointers enable two-way traversal.
