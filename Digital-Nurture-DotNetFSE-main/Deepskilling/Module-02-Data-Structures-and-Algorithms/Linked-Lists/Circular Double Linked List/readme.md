# Circular Double Linked List

## Introduction

A Circular Doubly Linked List is a linked list where each node contains previous and next references, and the first and last nodes are connected.

Example:

```text
10 ⇄ 20 ⇄ 30
↑           ↓
└───────────┘
```

## Operations

* Traversal
* Search
* Insert
* Delete

## Time Complexity

| Operation | Time Complexity |
| --------- | --------------- |
| Traversal | O(n)            |
| Search    | O(n)            |
| Insert    | O(1) or O(n)    |
| Delete    | O(1) or O(n)    |

## Advantages

* Bidirectional traversal
* Circular navigation
* Efficient insertion and deletion

## Learning Outcome

Learned how Circular Doubly Linked Lists support both forward and backward traversal while maintaining circular connectivity.
