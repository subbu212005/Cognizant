# Circular Linked List

## Introduction

A Circular Linked List is a linked list where the last node points back to the first node.

Example:

```text
10 → 20 → 30
↑         ↓
└─────────┘
```

## Operations

* Traversal
* Search
* Insert
* Delete

## Time Complexity

| Operation | Time Complexity |
| --------- | --------------- |
| Traversal | O(n)            |
| Search    | O(n)            |
| Insert    | O(1) or O(n)    |
| Delete    | O(n)            |

## Advantages

* Continuous traversal
* Efficient implementation of circular queues

## Learning Outcome

Learned the structure and applications of Circular Linked Lists.
