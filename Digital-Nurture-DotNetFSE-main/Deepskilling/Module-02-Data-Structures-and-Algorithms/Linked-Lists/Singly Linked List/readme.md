# Singly Linked List

## Introduction

A Singly Linked List is a linear data structure where each node contains data and a reference to the next node.

Example:

```text
10 → 20 → 30 → NULL
```

## Operations

* Traversal
* Search
* Insert
* Delete

## Time Complexity

| Operation           | Time Complexity |
| ------------------- | --------------- |
| Traversal           | O(n)            |
| Search              | O(n)            |
| Insert at Beginning | O(1)            |
| Insert at End       | O(n)            |
| Delete              | O(n)            |

## Advantages

* Dynamic size
* Efficient insertion and deletion

Learned the structure and operations of a Singly Linked List.
