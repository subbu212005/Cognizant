# Exercise 2 - Implementing the Factory Method Pattern

## Objective
Implement the Factory Method Design Pattern to create objects without specifying their exact classes.

## Concepts Used
- Factory Method Pattern
- Interfaces
- Polymorphism
- Object Creation

## Description
The Factory Method Pattern provides an interface for creating objects while allowing subclasses or factory classes to decide which object to instantiate. This pattern promotes loose coupling and makes the application easier to extend and maintain.
