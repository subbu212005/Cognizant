# Strategy Design Pattern

## Definition
The Strategy Pattern defines a family of algorithms, encapsulates them, and makes them interchangeable.

## Why Strategy?

- Allows changing behavior at runtime.
- Reduces conditional statements.
- Improves maintainability.

## Real World Example

Google Maps can choose different routes such as walking, driving, or cycling.

## Program Output

```
Payment made using Credit Card
```

## Conclusion

The Strategy Pattern allows selecting algorithms dynamically without modifying existing code.
